Movie Website

Registration/Log In


Before a User can see the Movielist, they will have to register a account

This is the required details to register a account

Upon Successful registration, they will be prompted to login

Once logged in, the username will be fetched from the SQL Database and displayed

Profile


After the User Log In, the Navigation bar will change to show the "Profile", "Movie List", "Add Review" and "Logout" Pages

When the user clicks the "Profile" Page, a short description of the user will be taken and displayed from the Username Database

MovieList


After a user is directed to the Movie List page, they have the option of searching for a movie

They have to type in the Movie's TItle

A Database will be scanned and if successful will show the movie being searched

Description


Once you choose the movie, you will be redirected to a page of the movie's description

the description is taken from the database

Movie Reviews


After the user has seen the description, the user can proceed to the page where it will list reviews

the reviews are for the specific Movie according to the Movie Id

In this example, the Movie Id for John Wick is 1

From the Navigation Bar, the user can add a review to the specific movie by the "Add Review" button

After doing so the New Review will be added into the Database and be shown in the Movie Reviews

The User will then have the option to either delete or Edit the Movie's Review

Logout


The Final Page is the "Logout" page, after the user clicks it, a prompt will tell the user if they want to confirm to logout

Once they do, they will be redirected to the Login Page and the Navigation Bar will return to only show the "Registration" , "Login" and "Movie List" Page
